瑜伽焰口施食法会录音点读版。

- **录音时间**：2024年4月30日
- **地点**：云南晋宁
- **文本**：《瑜伽焰口施食要集》

## 📖 使用方式

### 步骤一：下载文件

1. 下载本仓库的 `晋宁焰口标注工具.html`
2. 下载音频文件（两个MP3），放到同一文件夹：
   - [晋宁焰口上.mp3 下载链接（百度云）](https://pan.baidu.com/s/xxxx) — 需要法师提供
   - [晋宁焰口下.mp3 下载链接（百度云）](https://pan.baidu.com/s/xxxx)

### 步骤二：人工标注

1. 用浏览器打开 `晋宁焰口标注工具.html`
2. 点击 ▶ **开始标注**
3. **一边播放音频，一边点击对应的经文段落**
4. 听一段 → 点一段 → 直到全部标完
5. 点击 **💾导出标注结果**，复制JSON文本发给我

### 快捷键

| 按键 | 功能 |
|------|------|
| `Space` | 播放/暂停 |
| `Enter` | 标注当前选中的段落 |
| `↑ ↓` | 切换到上一个/下一个未标注的段落 |
| `← →` | 快退/快进5秒 |

### 步骤三：生成点读页面

我把您标注好的数据注入HTML，生成最终点读页面，推回到这个仓库。
