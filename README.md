# 晋宁焰口 · 瑜伽焰口施食要集点读标注工具 v2

瑜伽焰口施食法会录音点读版。

- **录音时间**：2024年4月30日
- **地点**：云南晋宁
- **文本**：《瑜伽燄口施食要集》（尊胜咒本，203页）

## 使用方式

### 在线使用（推荐）

打开在线地址：http://140.83.62.197:8080

### 本地使用

1. 下载本仓库
2. 确保两个音频文件在同一目录：
   - `audio-a.mp3`（上半部）
   - `audio-b.mp3`（下半部）
3. 运行 `node server.js` 或直接用浏览器打开 `index.html`（本地打开音频需支持跨域）

### 标注流程

1. **选择音频源**（上半部/下半部）
2. **播放音频**，眼睛看法本图片
3. **听到对应段落时，点击法本图片上的文字位置**
4. 在弹出的对话框中添加标签文字（可选），确认
5. 标记点自动记录当前播放时间
6. 以后点击标记点即可跳转播放

### 快捷键

| 按键 | 功能 |
|------|------|
| `Space` | 播放/暂停 |
| `←` `→` | 快退/快进 5 秒 |
| `↑` `↓` | 上/下一页 |
| `Ctrl+S` | 保存标记 |
| `Ctrl+E` | 导出标记 |

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 主页面（标注工具） |
| `server.js` | HTTP 服务器 |
| `pages/` | PDF 每页转图片（203页） |
| `audio-a.mp3` | 上半部录音 |
| `audio-b.mp3` | 下半部录音 |

## 数据存储

标记数据自动保存在浏览器 **localStorage** 中，更换设备或清除浏览器缓存会丢失。建议定期使用顶部 💾 导出按钮备份 JSON 文件。
